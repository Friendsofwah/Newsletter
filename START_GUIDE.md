## 🚀 How to Start the Application

### Option 1: Development Mode (Recommended for testing)
```bash
npm run dev
```
Then open your browser to `http://localhost:3000`

### Option 2: Production Mode
```bash
npm run build
npm run preview
```
Then open your browser to `http://localhost:4173`

### Option 3: Static File Serving
You can serve the `dist/` folder with any web server after running `npm run build`.

## 📋 First Steps After Starting

1. **Explore the Dashboard**: View the welcome screen with sample statistics
2. **Create Your First Newsletter**: Click "Create Newsletter" and choose a template
3. **Try the Templates**: Test "Monthly Industry Update", "Tax Season Special", or "Compliance Alert"
4. **Add Industry News**: Select relevant news items from the sidebar
5. **Preview Your Work**: Use the preview toggle to see formatted output
6. **Save and Manage**: Save as draft and view in the newsletters list
7. **Customize Settings**: Update your profile and preferences

## 🔧 Troubleshooting

If you encounter issues:

1. **Clear browser cache**: Hard refresh (Ctrl+F5 or Cmd+Shift+R)
2. **Check Node version**: Ensure Node.js 16+ is installed
3. **Reinstall dependencies**: Run `npm install` again
4. **Build issues**: Run `npm run build` to check for errors
